# AI-Powered Transcript-to-Notes System (YouTube Lectures)

An AI-driven application for transforming dense or rapid YouTube lecture streams into structured, textbook-quality study material using advanced transcription and LLM layers.

---

## Pre-requisites

* **Python 3.10+**: Install from [python.org](https://www.python.org/)
* **Git**: Install from [git-scm.com](https://git-scm.com/)
* **LLM API Key**: Create an account or get keys from an OpenAI-compatible provider

---
